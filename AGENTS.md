# Resume Project Rules

## 자동화된 에셋

- `assets/.automatic/` 폴더는 절대 직접 건드리지 않는다 (읽기, 수정, 삭제 모두 금지).
- 아이콘은 별도 스크립트(`scripts/download-icons`)로 관리된다. 수동으로 추가/제거하지 않는다.
- 새 아이콘 쿼리를 추가할 때는 반드시 Iconify API(`https://api.iconify.design/{prefix}/{name}.svg`)로 존재 여부를 확인한 후 사용한다.
- 프로젝트 구조를 어림짐작하지 않는다. 모르면 사용자에게 물어본다.
